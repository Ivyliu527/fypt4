**适用场合**：口头报告与原型示范（每组 20 分钟时段）

**要求摘要**：
- **简报表长**：不超过 **12 分钟**，投影片不超过 **20 张**
- **内容须含**：专案目标、问题背景、所提供的功能、部分高阶分析与设计图（如系统架构图、使用案例图、类别图等）、专案时程及进度摘要
- **简报表束后**：每位学生 **3 分钟** 展示其原型实作（须事先设定与测试系统）
- **报告与示范语言**：英语 | **服装**：正式
- **评分**（同等考量）：材料选择 · 准备 · 发表／处理问题

**说明**：以下共 **20 张** 投影片，每张标题 + 建议要点（3～5 条），可直接用于制作幻灯片。建议每张约 **30～40 秒**，总时长控制在 12 分钟内。

---

### 第 3 张：专案目标
- **主要目标**：打造一站式辅助应用，帮助视障用户更独立地生活
- **范围**：环境感知、阅读辅助、**出行协助（导航 + 可步行路径分析）**、紧急求助、语音控制、即时志愿者协助
- **成功标准**：语音优先、少触控；支持粤语、英语、普通话；无视力也可用（TTS、震动、大按钮）

---

### 第 4 张：问题背景 — 视障人群面临的生活问题
- **出行**：路牌、红绿灯、斑马线难以识别，易迷路、碰撞
- **环境**：难以识别周围的人、车、障碍物与光线
- **信息**：无法阅读文档、票据、货币面额，日常依赖他人
- **紧急**：难以及时一键求助、分享位置、联络家人／999
- **可用性**：多数 App 依赖视觉介面，视障用户难以独立操作

---

### 第 5 张：核心痛点与我们的应对
- 看不见周围 → 实时 AI 物体检测 + 语音播报（人/车/位置）
- 不会认路、怕走错 → 语音目的地 + 步行路线 + 分段 TTS 导航 + 可步行路径分析
- 读不了文件、认不了钱 → OCR + 港币识别 + 多语言朗读
- 紧急时难求助 → 长按 3 秒紧急按钮 + 选联络人 + 自动发位置/短信/拨号
- 不想一直点屏幕 → 语音命令 + 连续对话 + LLM + 手势快捷

---

### 第 6 张：App 定位与社会价值
- 瞳伴 = **综合智能辅助平台**（环境感知、阅读、出行、紧急、语音、即时协助）
- 技术支撑：计算机视觉 + 多语言语音 + 地图与导航
- **社会价值**：平等参与、安全与尊严、技术普惠、信息无障碍与智慧城市参考

## 第二部分：所提供的功能

---

### 第 7 张：功能总览（七大核心功能）
- 环境识别 | 阅读助手 | 语音助手 | 寻找物品
- 即时协助 | 出行协助 | 手势管理
- 首页统一入口 + 多语言（粤/英/普）+ 紧急求助长按入口

---

### 第 8 张：环境感知 · 文件与货币 · 紧急求助（概要）
- **环境感知**：摄像头 + SSD/YOLO → 多帧融合、人/车优先、左/中/右位置、夜间模式 → TTS 播报
- **文件与货币**：ML Kit OCR + 港币识别 → 朗读；独立阅读与确认找零
- **紧急求助**：长按 3 秒 → 选联络人 → 短信 + 位置 + 拨号（含 999）、电话号逐位朗读

---

### 第 9 张：出行协助 — 定位与价值
- **作用**：从「说目的地」到「跟着语音走」的全流程
- 路线规划 + 分段 TTS 导航 + 接近转弯提醒 + 偏离重规划
- **可步行路径分析**：前方是否畅通、需左/右绕行、斑马线/红绿灯提示
- **意义**：仅凭语音完成输入目的地到步行导航，减少碰撞与迷路

---

### 第 10 张：出行协助 — 技术实现
- **路线**：高德 Web 服务（地理编码 + 步行路线）+ NavigationController 定位与偏离重规划
- **导航页**：高德 MapView、TTS 分段播报「前行 XX 米」「左转」，约 20 米内自动播下一段
- **语音入口**：「我要去 XXX」→ VoiceCommandManager 解析 → 自动填入目的地
- **可步行路径**：CameraX + YOLO → WalkAssistManager 左/中/右分析 → 路况短语播报

---

### 第 11 张：语音命令与智能对话 · 其他功能（概要）
- **语音**：多语言 ASR、模糊匹配、「我要去 X」解析、LLM（DeepSeek/GLM-4-Flash）、不可用时回退关键词
- **其他**：即时协助（志愿者、视讯）、寻找物品、手势管理、系统设定（语速/音量/震动/多语言 TTS）

---

### 第 17 张：创新点与亮点总结
- **「听得懂的出行」全链路**：语音目的地 → 高德步行路线 → 分段 TTS + 转弯提醒 + 偏离重规划 + 可步行路径分析（不看地图也能安全走）
- **多帧融合 + 优先级**：连续 3 帧才播报，人/车优先、左中右位置；**夜间模式**自动低光检测与播报
- **语音与出行结合**：「我要去 X」直接进入导航；**无障碍贯穿**：大按钮、高对比、震动、电话逐位朗读、三语言、TalkBack

---

### 第 18 张：用户流程总览（一图流）
- 环境：主頁 → 環境識別 → 相機 → 語音播報
- 阅读：主頁 → 閱讀助手 → 對準 → 朗讀
- 出行：主頁 → 出行協助 → 開始出行 → 目的地 → 導航頁（路線 + 可步行路徑）
- 紧急：主頁 → 長按 3 秒 → 選聯絡人 → 發送/撥號
- 语音：主頁 → 語音助手 → 指令或連續對話
