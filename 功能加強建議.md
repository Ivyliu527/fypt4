### 3. **場景理解（室內/室外）** ⭐⭐⭐⭐
**優先級：中**

**功能描述：**
- 自動識別當前環境是室內還是室外
- 根據場景調整檢測策略（室內：家具、電子產品；室外：交通、建築）
- 提供場景描述（如："室內環境"、"室外街道"）

**實現方式：**
```java
// 根據檢測到的物體類型判斷場景
private String detectScene(List<DetectionResult> results) {
    int indoorObjects = 0;  // 家具、電子產品
    int outdoorObjects = 0; // 交通、建築
    
    for (DetectionResult result : results) {
        if (isIndoorObject(result.getLabel())) indoorObjects++;
        if (isOutdoorObject(result.getLabel())) outdoorObjects++;
    }
    
    if (indoorObjects > outdoorObjects) return "室內";
    return "室外";
}
```

### 4. **物體大小估算** ⭐⭐⭐
**優先級：中**

**功能描述：**
- 估算物體的相對大小（大、中、小）
- 結合距離信息，提供更準確的大小描述
- 對大型物體（如：車、門）特別標註

**實現方式：**
```java
private String estimateObjectSize(RectF bbox, String objectLabel) {
    float area = (bbox.right - bbox.left) * (bbox.bottom - bbox.top);
    
    // 根據物體類型和面積估算
    if (isLargeObject(objectLabel) && area > 0.1f) {
        return "大型";
    } else if (area > 0.05f) {
        return "中型";
    }
    return "小型";
}
```

### 7. **物體數量統計** ⭐⭐⭐
**優先級：中**

**功能描述：**
- 統計畫面中同一類物體的數量
- 提供更詳細的環境描述（如："檢測到3個人"）

**實現方式：**
```java
private Map<String, Integer> countObjects(List<DetectionResult> results) {
    Map<String, Integer> counts = new HashMap<>();
    for (DetectionResult result : results) {
        String label = result.getLabel();
        counts.put(label, counts.getOrDefault(label, 0) + 1);
    }
    return counts;
}
```
